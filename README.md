# Devops_90-Days


After leaving my previous role, I enrolled in a DevOps training program around April where I learned AWS, Linux, Docker, Kubernetes, Jenkins, and Terraform.

Initially, my focus was more on understanding concepts, but I realized that I needed stronger hands-on practice and real project experience to clear interviews.

So I reassessed my approach and started a structured 90-day plan where I am now focusing deeply on hands-on practice, command-level understanding, and building real-world projects using tools like AWS, Kubernetes, Terraform, Helm, ArgoCD, Prometheus, and Grafana.

This time, my focus is on practical implementation and end-to-end project building, which has significantly improved my confidence. I am now actively preparing and ready for DevOps roles.
